# Rent Well - A Real Estes Website

## Project Description
This website is designed for individuals seeking accommodation in a specific city as well as property owners looking to sell or rent their properties. It provides a user-friendly interface with detailed property listings, allowing users to make informed decisions before visiting a location in person. By offering comprehensive information and an intuitive design, this platform helps users shortlist properties from the comfort of their homes, saving both time and effort.

## Features
- **Rental Properties** – Browse available rental listings.
- **Selling Properties** – View properties available for sale.
- **Filter Page** – Apply various filters to refine property searches.
- **Home Page** – Access general property listings and website features.
- **User-Friendly Interface** – Intuitive design for seamless navigation.
- **Responsive Design** – Ensures a smooth experience on different devices.

## Technologies Used
- **HTML** – Structure of the website.
- **CSS** – Styling and layout.
- **JavaScript** – Interactive elements and functionality.
- **Bootstrap** – Responsive design and styling.
- **Custom Fonts** – Lato and Montserrat for enhanced typography.
